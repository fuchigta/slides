## プラグインの概要

Marp エコシステムを活用し、美しいスライドを効率的に作成するための<br>Claude Code プラグインです。

- テーマとレイアウトを組み合わせた宣言的なスライド記述
- Markdown ベースで、バージョン管理やコラボレーションが容易
- PDF / HTML / PPTX / PNG への一括エクスポートに対応
