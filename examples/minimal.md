## レイアウトシステムの概要

全レイアウトは3つのカテゴリに分類され、用途に応じて使い分けます。

- **Basic** — title-slide / section-divider / content-slide / two-column / image-slide
- **Business** — comparison-table / stats-grid / timeline / org-chart / quote-slide
- **Tech & General** — code-block / architecture-diagram / demo-slide / asymmetric / icon-grid / summary-slide
